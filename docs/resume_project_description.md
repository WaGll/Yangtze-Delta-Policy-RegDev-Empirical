# Resume Project Description

## 中文简历版本

**城市绿色全要素生产率、政策评估与空间计量研究**  
- 基于城市面板数据，构建数据清洗、变量生成、SBM-GML 测算、多期 DID 与空间计量的一体化复现 pipeline。  
- 使用 Python 完成缺失值插补、资本存量测算、单位统一、变量构建和包含非期望产出的 SBM-GML 测算。  
- 使用 Stata 完成多期 DID、平行趋势检验、安慰剂检验、空间权重矩阵、Moran 指数、空间杜宾模型、稳健性检验、异质性分析与中介机制检验。  
- 搭建符合学术复现标准的 GitHub 仓库，包含中英文 README、公式说明、数据目录规范和项目主页。

## English Resume Version

**Green Total Factor Productivity, Policy Evaluation and Spatial Econometrics**  
- Built a reproducible research pipeline for city-level panel data, covering data cleaning, variable construction, SBM-GML estimation, multi-period DID, and spatial econometric models.  
- Used Python to implement interpolation, capital stock measurement, unit harmonization, variable construction, and undesirable-output SBM-GML estimation.  
- Used Stata to conduct DID regressions, parallel trend tests, placebo tests, spatial weight matrix construction, Moran's I, spatial Durbin models, robustness checks, heterogeneity analysis, and mediation analysis.  
- Designed a publication-ready GitHub replication package with bilingual README files, formula notes, structured data folders, and an academic project webpage.
